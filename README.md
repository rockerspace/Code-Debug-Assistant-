# Code Debugging Assistant


A small full-stack app that helps developers debug pasted code: syntax checks, runtime explanations, suggested fixes and step-by-step debugging instructions.


Tech: React (Monaco), Node.js/Express, OpenAI Responses API.


## Features
- Syntax error detection (local parsers where available)
- LLM-driven runtime explanation & suggested fixes
- Step-by-step debugging guide
- Minimal, portfolio-ready UI
